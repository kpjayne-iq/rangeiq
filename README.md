# RangeIQ Poker

Interactive poker decision training — built by Between the Cards.

## Deploy to Vercel

1. Push this repo to GitHub
2. Connect to Vercel (vercel.com → Add New Project → select this repo)
3. Vercel auto-detects Vite — click Deploy
4. Add custom domain `rangeiqpoker.com` in project settings

## Local Development

```bash
npm install
npm run dev
```

## Update Workflow

1. Replace `src/RangeIQ.jsx` with the updated file
2. Commit to GitHub
3. Vercel auto-deploys in ~30 seconds

## Tech Stack

- React 18
- Vite 6
- Deployed on Vercel (free tier)
